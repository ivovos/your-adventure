# Educational Content Summary

## Overview
The story now contains **13 educational challenges** integrated throughout different story paths. Your child will encounter between 5-8 challenges depending on which choices they make, encouraging multiple playthroughs.

## Kent Test Question Format

All spelling challenges now use authentic 11+ Kent Test formats:

**Format 1: "Which spelling is CORRECT?"** (Used for 4 challenges)
- Shows 4 variations of the same word
- Student must identify the correctly spelled version
- No word shown in the question itself

**Format 2: "Which word is spelled INCORRECTLY?"** (Used for 2 challenges)
- Shows 4 different correctly/incorrectly spelled words
- Student must spot the ONE misspelled word
- Tests multiple words simultaneously
- Bonus learning: introduces additional words (BELIEVE, DEFINITELY)

This prevents cheating by copying and matches real test conditions!

## Kent Test Words Covered

### Spelling Challenges (8 words taught, 6 tested directly)

1. **NECESSARY** - One C, two S's
   - Memory trick: "One Collar, two Sleeves"
   - Common mistake: NECESARY, NECCESARY

2. **SEPARATE** - A in the middle
   - Memory trick: "There's A RAT in separate"
   - Common mistake: SEPERATE

3. **BELIEVE** - I before E rule (taught via Format 2)
   - Appears in "find the incorrect" challenge as BELEIVE
   - Memory trick: "I before E except after C"
   - Common mistake: BELEIVE

4. **DEFINITELY** - From DEFINITE (taught via Format 2)
   - Appears in "find the incorrect" challenge as DEFINATELY
   - Memory trick: "There's FINITE in definitely"
   - Common mistake: DEFINATELY

5. **OCCURRED** - Double R before -ED
   - Rule: Double consonant after short vowel
   - Common mistake: OCCURED

6. **DISAPPEAR** - One S, two P's
   - Structure: DIS + APPEAR
   - Common mistake: DISSAPEAR

**Bonus Words** (seen in "find incorrect" challenges):
- **FIERCE** - Correctly spelled option
- **RECEIVE** - Correctly spelled option
- **CEILING** - Correctly spelled option
- **ACCIDENTALLY** - Correctly spelled option

### Verbal Reasoning Challenges (7 words)

1. **CONCEALS** - Synonym: Hides
   - Opposite of: Reveals
   - Context: Hiding secrets

2. **PERMANENT** - Antonym of: Temporary
   - Meaning: Lasting forever
   - Context: Time duration

3. **ANCIENT** - Synonym: Old
   - Meaning: Very old, from long ago
   - Context: Historical items

4. **OBTAIN** - Meaning: Get or acquire
   - Synonym: Receive, gain
   - Context: Getting evidence

5. **SUFFICIENT** - Meaning: Enough, adequate
   - Context: Having the right amount
   - Related: Insufficient (not enough)

6. **SUSPICIOUS** - Meaning: Questionable, doubtful
   - Context: Untrustworthy behavior
   - Related: Suspicion, suspect

7. **EVIDENCE** - Meaning: Proof or information
   - Context: Detective work
   - Usage: Facts that prove something

## Educational Features

### Challenge Types
- **Multiple Choice**: All challenges use 4-option multiple choice format
- **Hints Available**: Every challenge has an optional hint
- **Explanations**: Detailed explanations provided after each answer
- **Retry Option**: Wrong answers can be retried (except for spelling)

### Learning Reinforcement
- **Memory Tricks**: Mnemonic devices for spelling words
- **Context Integration**: Words used naturally in the story
- **Varied Question Types**: Synonyms, antonyms, definitions, spelling
- **Progressive Difficulty**: Earlier challenges are slightly easier

### Engagement Features
- **Dice Rolls**: Add excitement and unpredictability (2 total)
- **Multiple Paths**: Encourages replaying to see different content
- **Detective Theme**: Makes learning feel like solving a mystery
- **Immediate Feedback**: Clear right/wrong with explanations

## Tips for Parents/Tutors

1. **Encourage Multiple Playthroughs**: Different paths teach different words
2. **Discuss Words After**: Review the victory screen's word list together
3. **Use Memory Tricks**: The mnemonics are specifically designed for Kent Test
4. **Note Patterns**: Point out spelling patterns (doubling consonants, etc.)
5. **Apply in Context**: Use these words in daily conversation
6. **Track Progress**: Each playthrough takes 10-15 minutes
